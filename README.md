# Resume Project

## Description
## Description
Resume Analyzer for Students is a Python-based project designed to help students evaluate and improve their resumes.  
The system analyzes the resume content and provides feedback based on basic ATS (Applicant Tracking System) rules such as keyword matching, formatting, and skill relevance.  

This project helps students understand how their resume performs in real-world job screening systems and guides them to make improvements for better internship and job opportunities.

## Technologies Used
- Python

## How to Run
Open terminal and run:

python app.py
